# Bash Automation

This folder contains Bash scripts for Linux administration, DevOps, SRE, container operations, monitoring and security-related automation.

The scripts are focused on practical operational tasks such as system checks, log analysis, Docker maintenance, Kubernetes helpers, backups, process monitoring and basic security validation.

- Linux administration
- System health checks
- Docker automation
- Kubernetes helper scripts
- Log analysis
- Backup and cleanup tasks
- Monitoring and alerting
- Security checks
- File and permission audits
- Network diagnostics

# Requirements

Most scripts are designed for Linux-based environments and require Bash.
